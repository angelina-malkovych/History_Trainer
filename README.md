# 📜 History Trainer

Інтерактивний тренажер для підготовки до НМТ з історії України.

## ✨ Можливості

- 📚 Вивчення історичних подій за допомогою карток
- 📝 Тестування знань дат
- ⭐ Система очок
- 📊 Профіль користувача
- ❗ Робота над помилками
- 📜 Історичний інтерфейс у стилі пергаменту

## 🛠 Технології

- Python 3
- Tkinter
- JSON

## 🚀 Як запустити

1. Клонувати репозиторій:

```bash
git clone https://github.com/angelina-malkovych/History_Trainer.git
```

2. Перейти в папку проєкту:

```bash
cd History_Trainer
```

3. Запустити програму:

```bash
python main.py
```
